# ️ Spartan Hub

**Spartan Hub** 是一款专为《光环：无限》（Halo Infinite）玩家打造的现代化 UWP 战绩查询工具。它基于 C# 开发，拥有原生流畅的 Windows 10/11 界面体验，帮助你随时随地查看自己的斯巴达服役档案、竞技阶级（CSR）以及详细的战斗统计数据。

本项目完全开源，核心数据由 [GravlLift/halo-infinite-api](https://github.com/GravlLift/halo-infinite-api) 强力驱动。

## 快速开始 (Getting Started)

### 环境要求

* Windows 10 (版本 1903 或更高) 或 Windows 11
* .NET Framework / .NET Core (根据你的项目实际版本填写，例如 .NET 6.0)

### 安装与运行

1. **克隆仓库到本地：**
   
   ```bash
   git clone https://github.com/Pigeon-Ming/SpartanHub.git
   ```

2. **使用 Visual Studio 打开解决方案：**
   双击根目录下的 `SpartanHub.sln` 文件。

3. **还原 NuGet 包并运行：**
   在 VS 中还原所有 NuGet 依赖包，按下 `F5` 即可编译并启动应用。

## ️ 技术栈 (Tech Stack)

* **开发语言**：C#
* **应用平台**：UWP (Universal Windows Platform)
* **UI 框架**：XAML
* **数据接口**：[GravlLift/halo-infinite-api](https://github.com/GravlLift/halo-infinite-api)
